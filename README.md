# English Reading and Writing Practice — Landing Page

A premium, modern landing page for the academic English book
**English Reading and Writing Practice** by N.N. D, A.F. A and Sh.Sh. U.

Built with **Vite + React + Tailwind CSS + Framer Motion + Lucide React**.

## Quick start

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

To produce a production build:

```bash
npm run build
npm run preview
```

## Structure

```
src/
  App.jsx                Top-level layout
  index.css              Tailwind + custom theme
  components/
    Navbar.jsx           Sticky glass navigation
    Hero.jsx             Hero with 3D book mockup
    BookMockup.jsx       Interactive 3D / tilt mockup
    Authors.jsx          Author cards
    About.jsx            Light-themed feature grid
    ContentPreview.jsx   Lesson cards
    Statistics.jsx       Animated counters
    Testimonials.jsx     Reviews bento
    Gallery.jsx          Interactive book-spread carousel
    DownloadCTA.jsx      Final call-to-action
    Footer.jsx           Academic-style footer
    SectionHeading.jsx   Shared section heading
```
